
# Payment Form Test Cases

This repository contains test cases for a payment form.

## Scope
The following types of testing are covered:
- Functional testing (positive and negative scenarios)
- UI validation
- Usability testing
- Security testing
- Compatibility testing (cross-browser and cross-device)

## Files
- TestCase.pdf – detailed test cases for payment form validation

## Notes
The test cases include validation of payment processing, input fields, error handling, and basic security checks.
